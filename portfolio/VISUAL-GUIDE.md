## 🎓 How to Make Changes

**To add a new skill:**
1. Find Skills section in index.html
2. Copy one skill card
3. Paste it next to others
4. Update emoji and text
5. Save and refresh

**To add a new project:**
1. Find Projects section in index.html
2. Copy one project card
3. Paste next to others
4. Update content (title, description, link)
5. Save and refresh

**To change colors:**
1. Use Find & Replace
2. Search: `blue-` → Replace: `emerald-` (or your color)
3. Save and refresh
