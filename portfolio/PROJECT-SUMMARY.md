## 🆘 Troubleshooting

### Site doesn't look right
**Solution**: 
- Clear browser cache (Ctrl+Shift+Delete)
- Try different browser
- Check for typos in HTML

### Images don't show
**Solution**:
- Verify image is in `assets/images/` folder
- Check file name matches exactly (case-sensitive)
- Verify path is relative: `assets/images/pic.jpg` (not `/assets/`)

### Mobile menu doesn't work
**Solution**:
- Check JavaScript is allowed in browser
- Refresh page
- Try different browser
- Check browser console for errors (F12)

### Links don't work
**Solution**:
- Verify URLs include `https://`
- Check email link is formatted: `mailto:email@example.com`
- Test each link before deploying

### Slow loading
**Solution**:
- Compress images (TinyPNG.com)
- Check internet connection
- Use Lighthouse (Chrome DevTools) to audit

For more help, see [DEPLOYMENT.md](DEPLOYMENT.md) troubleshooting section.
